# Arbejdstidsberegner

En simpel webapp til hurtigt at beregne hvornår du skal gå hjem.

**[→ Åbn beregneren](https://eletrifiedcanibaldeer.github.io/arbejdstid/index.html)**

## Funktioner

- Indtast mødetid og se hvornår du skal gå hjem (7t 24min)
- Tillæg for kørsel til/fra arbejde (+15 min hver)
- Husker seneste indstillinger i browseren
- Fungerer som webapp på iPhone (Føj til hjemskærm)
